RowCreatorProducer

trait RowCreatorProducer[A, +B <: RowCreator[A]] extends AnyRef

SpecProducer is used to create different kinds of RowCreator instances. Classes that extend SpecProducer should only have zero-arg constructors. This is because Spec instances should only be parametrized by the JSON specification. Otherwise, one JSON specification could produce non-equivalent Spec instances in different environments. It is a design goal for this not to happen.

A

type of input passed to the spec.

B

implementation of the Spec[A] that is returned by the getSpec function.
